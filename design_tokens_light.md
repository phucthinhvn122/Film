
# Design Tokens - RoPhim Light Theme

This document outlines the new light theme color palette. The brightness of the original dark theme has been increased by approximately 20-30%.

## 1. Colors (Light Theme)

### Primary Palette
- `primary-bg`: `#F0F2F5` (Main background - a very light gray)
- `surface-1`: `#FFFFFF` (Slightly lighter surface, used for cards, drawer)
- `surface-2`: `#F8F9FA` (Elevated surface)
- `surface-3`: `#E9ECEF` (Component background, e.g., episode buttons)

### Accent Palette
- `accent-gold`: `#D4AC0D` (Primary gold, slightly desaturated for light theme)
- `accent-gold-darker`: `#B8860B` (Darker gold for gradients)
- `accent-blue`: `#2980B9` (Used for the rating button)
- `accent-red`: `#C0392B` (Used for close buttons)

### Text Palette
- `text-primary`: `#212529` (Primary text, dark gray)
- `text-secondary`: `#495057` (Secondary text, for subtitles, metadata)
- `text-muted`: `#6C757D` (Muted text, for less important details)
- `text-dark`: `#FFFFFF` (Text on dark backgrounds)

### Gradient Palette
- `gradient-han-quoc`: `linear-gradient(135deg, #A9CCE3, #FADBD8)`
- `gradient-trung-quoc`: `linear-gradient(135deg, #A3E4D7, #F5B7B1)`
- `gradient-hoat-hinh`: `linear-gradient(135deg, #FADBD8, #FDEDEC)`
